# Таланты для Мифик+

{% include resto_talents.html active="3112033" %}

Билд для Мифик+ мало чем отличается от рейдового, но есть несколько вариантов для усиления отхила по одной цели:

• **15 уровень:** Если вы часто используете {{ site.data.spells.hw }} или {{ site.data.spells.hs }}, то {{ site.data.spells.undulation }} будет отличным выбором, так как она очень сильно усиливает наш отхил по одной цели.  

• **25 уровень:** {{ site.data.spells.deluge }} дополнительно усилит отхил по одной цели, а {{ site.data.spells.eote_resto }} универсальный и удобный вариант. {{ site.data.spells.soe }} вообще не стоит брать, этот талант крайне слаб и проигрывает двум другим вариантам.  

• **30 уровень:** Тир вспомогательных талантов, выбирайте на ваше усмотрение.  

• **45 уровень:** {{ site.data.spells.ff }} отлично сочетается с {{ site.data.spells.deluge }}, усиливая наш отхил одной цели. Но в таком случае мы теряем бурст отхил от {{ site.data.spells.cbt }}.  

• **50 уровень:** {{ site.data.spells.high_tide }} все еще хороший выбор, но для Мифик+ {{ site.data.spells.asc_resto }} обычно полезнее.  

# Разбор талантов

## Первый тир, 15 уровень

* {{ site.data.spells.torrent }} – увеличивает **первоначальный** отхил способности {{ site.data.spells.riptide }} на **20%**.

Самый простой в использовании талант первого тира, который пассивно увеличивает первоначальное исцеление от {{ site.data.spells.riptide }}.

Сейчас {{ site.data.spells.riptide }} используется практически по кулдауну, так как первоначальное исцеление этой способности было усилено на **75%** по сравнению с BFA. Но сам талант был ослаблен, так что сейчас его следует выбирать только в том случае, если вы новичок и вам нужно как можно меньше кнопок, а использование {{ site.data.spells.ul }} вызывает сложности.

* {{ site.data.spells.undulation }} – усиливает на **50%** каждое третье применение {{ site.data.spells.hs }} или {{ site.data.spells.hw }}.

Хороший талант для Мифик+ и рейдовых боссов, где требуется сосредоточиться на исцелении одиночных целей. Дает очень сильное исцеление в одну цель, особенно при использовании зарядов {{ site.data.spells.tw22 }}.

* {{ site.data.spells.ul }} – активная способность. Лечит цель и дает бафф на **35%** отхила следующего заклинания прямого исцеления. Время восстановления **15** секунд.

Единственный талант первого тира, усиливающий наше АоЕ-исцеление, благодаря взаимодействию с {{ site.data.spells.ch }}. Наиболее полезен в рейде, где мы используем {{ site.data.spells.ch }} гораздо чаще, чем в Мифик+.

Усиливает только заклинания прямого исцеления, то есть не работает на {{ site.data.spells.{{ site.data.spells.downpour }} }} и {{ site.data.spells.{{ site.data.spells.wellspring }} }}. В отличие от таланта {{ site.data.spells.torrent }}, при использовании на {{ site.data.spells.riptide }} увеличивает не только первоначальное исцеление, но и его ХоТ часть.

В руках опытного игрока, знающего когда и как использовать данный талант, {{ site.data.spells.ul }} становится мощнейшим инструментом.

## Второй тир, 25 уровень

* {{ site.data.spells.eote_resto }} – дает способностям {{ site.data.spells.riptide }}, {{ site.data.spells.hst }} / {{ site.data.spells.cbt }} и {{ site.data.spells.lvb }} второй заряд.

Невероятно сильный талант, с которым гораздо сложнее пропустить использование  {{ site.data.spells.riptide }}, что в свою очередь приводит к генерации большего количества баффов {{ site.data.spells.tw22 }} за бой, что по цепочке приводит к увеличению эффективности используемых {{ site.data.spells.hs }} и ускорению применения {{ site.data.spells.hw }}.

Позволяет придержать сразу 2 {{ site.data.spells.riptide }} для моментов с перебежкой, а также минимизировует время простоя {{ site.data.spells.cbt }}.

Также стоит забывать о дополнительном заряде для способности {{ site.data.spells.lvb }}. На исцеление он эффект не окажет, но даст больше возможностей для нанесения урона в спокойные моменты боя, что очень важно в Мифик+ и на нередко встречающихся дпс-чеках в рейде.

* {{ site.data.spells.deluge }} – увеличивает исцеление ваших {{ site.data.spells.ch }}, {{ site.data.spells.hw }} и {{ site.data.spells.hs }} на **20%** для целей, находящихся в радиусе действия вашего {{ site.data.spells.rain }} либо под действием {{ site.data.spells.riptide }}.

Высокая стоимость {{ site.data.spells.ch }} и необходимость соблюдения условий для работы этого таланта делает его не лучшим выбором для рейда. Может пригодиться в боях, где большую часть времени рейд стоит кучно.

Этот талант можно реализовывать в Мифик+, но это приведет к снижению потенциального урона за счет отсутствия таланта {{ site.data.spells.eote_resto }}. Может быть полезен для увеличения отхила одной цели. 

Если ваша группа или рейд не стоит в {{ site.data.spells.rain }}, то {{ site.data.spells.deluge }} резко теряет свою эффективность. Если вы его использете, то обратите внимание ваших сопартийцев на этот момент.

* {{ site.data.spells.soe }} – поглощает **3** заряда активного {{ site.data.spells.eshield }} и исцеляет максимум **3** союзников, находящихся в непосредственной близости от текущей цели {{ site.data.spells.eshield }}.

На данный момент этот талант не имеет практической реализации из-за крайне низкого радиуса действия и необходимости гораздо чаще тратить время на обновление {{ site.data.spells.eshield }}. Худший выбор в этом тире для любого вида контента.

## Третий тир, 30 уровень

* {{ site.data.spells.spirit_wolf }} – защитная способность, снижающая входящий урон и увеличивающая скорость передвижения на **20%** после **4** секунд использования Ghost Wolf.

Является выбором по умолчанию для всех видов контента за исключением редких специфических ситуаций.

* {{ site.data.spells.egt }} – любой противник (кроме боссов) проходящий через область действия тотема сначала обездвиживается на **8** секунд, а после спадения эффекта обездвиживания замедляется на **50%**. Длительность тотема **20** секунд. Время восстановления **30** секунд.

Используется в ситуациях, когда противника требуется подержать в стороне от группы какое-то время. Пригодится против тех противников, которых невозможно оглушить. Достаточно сильная способность для кайта групп противников, может найти применение в Мифик+.

* {{ site.data.spells.static }} – сокращает время восстановления Capacitor Totem в зависимости от количества пораженных им противников.

Хороший выбор для Мифик+, если вашей группе не хватает способностей контроля.

## Четвертый тир, 35 уровень

* {{ site.data.spells.vigor }} – увеличивает максимальный запас здоровья целей ваших способностей прямого исцеления на **10%**.

Предпочтительный талант для использования в М+. Должен все время поддерживаться на танке, а в опасные моменты боя и на остальных членах группы. Во многих случаях, особенно в ключах высокого уровня, разница в **10%** здоровья означает разницу между жизнью и смертью.

В рейдах данный талант также имеет право на жизнь, особенно на энкаунтерах с опасными дебафами, применяющимися к одиночным целям и при невозможности эффективно реализовать другие таланты данного тира.

Помните, что эффект применяется к каждой цели способности {{ site.data.spells.ch }}, что упрощает его распространение по группе, особенно по целям с наименьшим количеством здоровья. 

* {{ site.data.spells.ewt }} – тотем, поглощающий часть урона, получаемого игроками в зоне его действия.

Поглощение работает на все источники урона, включая доты. Суммарно тотем может поглотить объем урона, равный максимальному запасу здоровья шамана его установившего.

Дает большое количество исцеления за одно ГКД, что делает его весьма сильным талантом для энкаунтеров, где у рейда есть возможность держаться кучно и нет необходимости в постоянном перемещении.

Может использоваться как мини-сейв для танка, в момент когда его атакуют несколько противников одновременно, так как тотем не имеет внутреннего кулдауна на срабатывание и ограничения по количеству источников урона или срабатываний для одного игрока.

Помните, что данный талант не очень эффективен против редких, но сильных атак босса, так как объем поглощения от одной атаки имеет ограничения, связанные с вашей силой заклинаний.

Единственный талант в этом тире, который повышает показатель вашего HPS.

* {{ site.data.spells.apt }} – воскрешает первого игрока, погибшего в радиусе действия этого тотема, с **20%** от максимального запаса здоровья. Не идет в счетчик боевых воскрешений. Кроме того, все игроки в зоне действия тотема получают дополнительные **10%** к запасу здоровья.

Уникальный и крайне мощный инструмент в арсенале Шамана Исцеления. Тотем не защищает от урона, который превышает **200%** от максимального здоровья игрока, в этом случае воскрешение от тотема будет недоступно. Крайне сильный талант для освоения и для особых механик, когда легче позволить союзнику умереть и воскресить его, чем тратить ману, кулдауны и время на его спасение.

За счет эффекта повышения здоровья союзников на **10%**, использование тотема не будет бесполезным даже в том случае, если никто не погиб за время его действия. 

## Пятый тир, 40 уровень

* {{ site.data.spells.ng }} – исцеляет вас на **20%**, когда уровень вашего здоровья падает ниже **35%** от максимального запаса.

Этот талант может спасти от смерти в том случае, если количество избыточного урона не превысило количество исцеления от таланта. Талант достаточно силен, но его срабатывание не всегда получается контролировать, так как он имеет внутренний кулдаун в **45** секунд. Берется по умолчанию, если не требуется дополнительное утилити от других талантов тира.

* {{ site.data.spells.gs }} – сокращает время перезарядки {{ site.data.spells.swg }} и повышает скорость передвижения при активации этой способности на **20%**.

Отличный выбор для любого энкаунтера, где нужно много передвигаться.

* {{ site.data.spells.wrt }} – устаналивает тотем в указанном месте, увеличивающий скорость передвижения любого союзника, зашедшего в зону его действия, на **60%** в течение **5** секунд.

Игрок повторно зашедший в область действия тотема обновит время действия баффа. Тотем можно разместить по ходу движения группы.

Это очень сильная способность, крайне полезная в боях где от группы требуется быстро переместиться из точки А в точку Б, ценность которой растет при отсутствии в вашей группе друида, способного ускорить группу при помощи {{ site.data.spells.stampeding_roar }}.

## Шестой тир, 45 уровень

* {{ site.data.spells.ff }} – использование заряда {{ site.data.spells.tw22 }} сокращает время применения вашего следующего заклинания на **20%**.

Этот талант может быть полезен в Мифик+, особенно для отхила одной цели. Данный талант работает с любыми заклинаниями, в том числе с {{ site.data.spells.rain }} и {{ site.data.spells.wellspring }}, что делает его более универсальным.

Не рекомендуется использовать его с {{ site.data.spells.hs }}, так как вы рискуете упереться в ГКД и не получить никакой выгоды от {{ site.data.spells.ff }}. Данный талант позволяет нам сохранить темп игры получаемый от таланта второго тира {{ site.data.spells.eote_resto }}, заменив его на {{ site.data.spells.deluge }} и увеличив таким образом исцеления по танку и группе в м+.

* {{ site.data.spells.downpour }} – это сильное АОЕ заклинание с плавающим временем восстановления. 

За счет особой механики кулдауна этого заклинания, не так страшно промахнуться им и задеть меньше союзников, чем необходимо. Однако частое использование ведет к большим потерям маны.

Может использоваться в рейдах либо в Мифик+ на неделях, требующих большого количества АОЕ исцеления.

* {{ site.data.spells.cbt }} – собирает **30%** от всего производимого вами исцеления и распределяет его между всеми ранеными союзниками в радиусе **40** метров от шамана, установившего тотем. Отхил срабатывает после уничтожения тотема, путем повторного применения способности или автоматически через **15** секунд после установки. 

Великолепный универсальный талант, лучший выбор в этом тире для большинства ситуаций.

Не собирает лечение от других тотемов, например {{ site.data.spells.htt }}. Но он собирает избыточное исцеление, однако **не стоит специально оверхилить для зарядки тотема**. 

Для максимальной эффективности старайтесь использовать все короткие кулдауны, такие как {{ site.data.spells.riptide }} и {{ site.data.spells.rain }}, до активации {{ site.data.spells.cbt }}. Лучший вариант зарядки тотема – использовать АОЕ-способности во время его действия.

Тотем всегда выбирает своими целями раненых союзников, равномерно распределяя лечение между ними, а значит его «взрыв» может быть использован как сильное бурст-исцеление танка (при отсутствии других раненых игроков в группе). 

Выбор таланта {{ site.data.spells.eote_resto }} во втором тире дает {{ site.data.spells.cbt }} второй заряд, что упрощает его использование, а также снижает время простоя этого таланта. Плюс это позволяет придерживать {{ site.data.spells.cbt }} к определенным моментам боя, не теряя время его восстановления впустую.

## Седьмой тир, 50 уровень

* {{ site.data.spells.high_tide }} – пассивный эффект, усиливающий **2** применения способности {{ site.data.spells.ch }} на **25** секунд за каждые **20 000** потраченной маны. При этом лечение от {{ site.data.spells.ch }} увеличивается на **10%** и его эффект не ослабевает при каждом перескоке.

По сути это просто пассивный кулдаун, который поначалу может показаться непривычным за счет своей механики срабатывания.

Лучше всего этот талант показывает себя на боях с большим количеством АОЕ-урона по группе, когда у игроков есть возможность держаться на небольшом расстоянии друг от друга и Вы уверены, что  {{ site.data.spells.ch }} будет использовано на максимальное количество целей. 

* {{ site.data.spells.wellspring }} – создает большую волну, распространяющуюся и исцеляющую союзников в широком конусе от шамана. 

Этот талант может быть полезен в боях, где у рейда нет возможности сгруппироваться в одном-двух местах и такие способности как {{ site.data.spells.ch }} и {{ site.data.spells.rain }} неэффективны. 

Время восстановления в **20** секунд отлично сочетается с талантом {{ site.data.spells.cbt }}. Однако некоторые неочевидные особенности механики данного таланта и отсутствие взаимодействия с другими пассивными способностями шамана делают его не простым в использовании и требуют опыта и сноровки.

Если кратко, эффективное исцеление от этого таланта имеет ограничение в **6** целей, а все избыточное исцеление от него снижается на **25%**. Подробнее читайте [в этом разделе](https://stormkeeper.ru/resto/rotation.html#%D1%80%D0%BE%D0%B4%D0%BD%D0%B8%D0%BA).

* {{ site.data.spells.asc_resto }} – сильная «бурст» способность со временем восстановления **3** минуты. 

Этот талант рекомендуется использовать в подземельях, так как он позволяет преобразовать исцеление одной цели в АоЕ отхил. Лучшее время для применения – моменты с сильным АОЕ-уроном по группе. 

В рейде данный талант берется на энкаунтерах с прогнозируемым уроном, когда требуются дополнительные кулдауны в определенные моменты боя. Однако стоит помнить, что радиус действия у таланта **20** метров, а значит рейд должен держаться более-менее кучно.

Отлично сочетается с талантом {{ site.data.spells.cbt }}, не забывайте его использовать перед применением {{ site.data.spells.asc_resto }}.
